# Исправление ошибок деплоя на Vercel

## Что делать если видите ошибку

### 1. Найдите полный текст ошибки

В логах Vercel прокрутите вниз до конца и найдите строку с ошибкой. Обычно она выглядит так:
- `Error: ...`
- `Type error: ...`
- `Module not found: ...`

### 2. Типичные ошибки и решения

#### Ошибка: "Module not found" или "Cannot find module"
**Решение:** Проверьте импорты в коде. Убедитесь, что все пути правильные.

#### Ошибка: "Type error" или TypeScript ошибки
**Решение:** 
- Проверьте типы в файлах `.ts` и `.tsx`
- Убедитесь, что все типы правильно импортированы

#### Ошибка: "DATABASE_URL not found"
**Решение:** 
- Vercel Dashboard → Settings → Environment Variables
- Добавьте `DATABASE_URL` (создайте базу данных в Storage → Create Database → Postgres)

#### Ошибка: "Prisma Client not generated"
**Решение:** 
- Убедитесь, что `prisma/schema.prisma` правильно настроен
- Проверьте, что `provider = "postgresql"` (не "sqlite")

### 3. Временное решение для сборки

Если нужно быстро задеплоить, можно временно отключить проверку типов:

В `next.config.js`:
```js
typescript: {
  ignoreBuildErrors: true,
},
eslint: {
  ignoreDuringBuilds: true,
},
```

**НО:** Это только для временного деплоя! Потом нужно исправить ошибки.

### 4. Проверка перед деплоем

Перед загрузкой на GitHub проверьте локально:
```bash
npm run build
```

Если сборка проходит локально, но падает на Vercel - проблема в переменных окружения.

## Что нужно сделать СЕЙЧАС

1. **Скопируйте полный текст ошибки** из логов Vercel (прокрутите до конца)
2. **Проверьте переменные окружения** в Vercel Dashboard:
   - `DATABASE_URL` - должна быть настроена
   - `JWT_SECRET` - должна быть настроена
3. **Загрузите обновленные файлы** на GitHub:
   - `tsconfig.json`
   - `next.config.js`
   - `prisma/schema.prisma`

## Быстрая проверка

Выполните локально:
```bash
npm install
npm run build
```

Если локально работает, а на Vercel нет - проблема в переменных окружения или конфигурации Vercel.
